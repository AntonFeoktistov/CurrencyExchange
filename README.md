# Currency Exchange

Проект реализует сервис для взаимодействия с валютами и их обменными курсами. 

## Возможности

- Просмотреть список доступных валют
- Добавить новую валюту
- Просмотреть доступные обменные курсы
- Изменить и добавить новые обменные курсы
- Рассчитать стоимость конвертации определенного количества одной валюты в другую

## Архитектура

### Controller 
Обрабатывает запросы, формирует и отправляет ответы. 

Класс BaseHandler обрабатывает входящие запросы и в зависимости от типа запроса перенаправляет действие в классы 
GetHandler, PostHandler, PatchHandler. 

Фунционал отправки JSON-ответов и взаимодействия с фронтендом вынесен в отдельный класс 
FrontMixin, от которого наследуются все хендлеры. 

## Service 
Осуществляет логический функционал проекта, обрабатывает ошибки. Методы валидации данных вынесены в класс Validator. 
Сервис обрабатывает информацию, входящую от хендера и исходящую из базы данных, формирует данные ответа и перенаправляет их в контроллер. 


## Model 
Реализует взаимодействие с базой данных. В родительском классе BaseModel реализован метод подключения к бд. Методы для 
взаимодействия с таблицей валют Currency содержатся в классе CurrencyModel, для взаимодействия с таблицей обменных курсов - в классе 
ExchangeModel. 

Также в слое Model содержится класс Serializer, отвечающий за формирование данных формата JSON для передачи в Service. 

## View
Формирует JSON ответы для Controller 

## Доступные запросы 

### Валюты

- `GET /currencies` — список всех валют
- `GET /currency/{code}` — получение валюты по коду
- `POST /currencies` — регистрация новой валюты (`x-www-form-urlencoded`: `name`, `code`, `sign`)

### Курсы обмена

- `GET /exchangeRates` — список всех обменных курсов
- `GET /exchangeRate/{USDEUR}` — получение обменного курса конкретной пары валют (USDEUR - пример формата ввода)
- `POST /exchangeRates` — регистрация нового обменного курса. Параметры принимаются в формате `x-www-form-urlencoded`: `baseCurrencyCode`, `targetCurrencyCode`, `rate`
- `PATCH /exchangeRate/{USDEUR}` — обновление существующего курса 

### Конвертация валют

- `GET /exchange?from=BASE&to=TARGET&amount=AMOUNT` — расчет суммы в целевой валюте (`TARGET`) для перевода указанного количества средств (`AMOUNT`) из исходной валюты (`BASE`) по текущему курсу  
  Поддерживаются:
  - прямой курс;
  - обратный курс;
  - кросс-курс через USD.
